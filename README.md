# Fika Oraklet

Ett litet testprojekt for att ova ett enkelt GitHub-flode mellan flera datorer.

## Vad ar det?

En liten statisk webbapp som slumpmassigt valjer dagens fika med maximal sjalvsakerhet och minimal vetenskap.

## Filer

- `index.html` - strukturen
- `style.css` - layout och utseende
- `script.js` - logiken for oraklets svar
- `PROJECT_STATUS.md` - nulage
- `TODO.md` - nasta steg

## Kor lokalt

Eftersom det ar en statisk sida kan du bara oppna `index.html` i webblasaren.

## Tanken med projektet

Det har repot ar medvetet enkelt. Malet ar inte appen i sig, utan att ge dig ett rent, begripligt startflode for:

1. ha ett projekt i GitHub
2. jobba pa flera datorer
3. lata projektets viktiga kontext ligga i filer, inte bara i chatten
